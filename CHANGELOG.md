# Changelog

All notable changes to the "Icon Preview" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-06

### Added
- Initial release
- Preview image files as their own thumbnails in VS Code Explorer
- Support for png, jpg, jpeg, gif, bmp, ico, svg, webp formats
- Auto-activation of icon theme
- Configurable image extensions and icon size
- Auto-refresh when image files change
- Manual refresh command
- Full undo/redo support
